
СПРАВКА MOONSHELL
------------------------------

Встроенные команды с их синтаксисом:
    help [~com]             - показать справку
    exit                    - выход
    ls                      - показать содержимое папки
    copy [scr] [cp_to]      - скопировать файлы
    touch [scr] [text]      - создать файл и напечатать текст в него
    cd [path]               - сменить папку
    pwd                     - показать текущую папку
    mkdir [path]            - создать новую папку
    rm [path]               - удалить папку или файл
    mv [scr] [mv-to]        - переименовать файл или папку
    clear                   - отчистить вывод
    print [text] [path_p]   - напечатать на принтере текст
    sudo [com]              - запустить команду от имени суперпользователя
    echo [text]             - напечатать текст на экране
    
Подробнее про каждую команду ва можете узнать тут:
    $ help [команда]
